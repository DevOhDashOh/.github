  - 🚩[깃 설치](https://taewow.tistory.com/13) 👉 깃허브 데스크톱, Gitlens 등을 활용하면 더 편합니다.

  - 아래의 Flow가 이해되면 충분합니다.
  ```.shell
  Fork --> Clone --> branch 생성 --> 코드 작성 --> add --> commit --> push --> PR --> Code Review --> Merge  
  ```
